# Git常用命令

### 本地库初始化：git init

### 设置签名：

##### git config --global user.name 设置全局用户名

##### git config --global user.email 设置全局邮箱名

###### 保存位置 ~/.gitconfig

### 提交文件

- git status 查看文件状态
- git add 添加文件到暂存区
- 1.git commit 提交文件到本地库 2.git commit -m "描述内容"
- git remote add origin 添加远程库的别名
- git push origin master 添加到远程库
